Простой проигрыватель мелодий на ATTINY13. Схема находится в файле s.gif.
Устройство может проигрывать ноты 1, 2, и частично 3 октавы длительностью 1, 2, 4, 8, 16, 32, 64.
Ноты мелодии находятся в байтовом массиве в флэш памяти и воспроизводятся последовательно. Формат массива следующий:
{
nC2 + dX32,
nA1 + dX16,
nSL + dX16,
..
}
где nC2 и nA1 - обозначение нот До второй октавы и Ля первой октавы. dX32 продолжительность звучания ноты nC2 равная 32, а dX16 - ноты nA1 равная 16 соотвественно. Минимальная продолжительность по умолчанию определена 60мс.
Нота nSL обозначает паузу, длительность задается также как и у остальных нот.
Устройсво расчитано на работу с пьезоизлучателем звука, если будет использоватся электромагнитный, то желательно на выход поставить транзистор.
Фьюзы для тестовой прошивки LOW = 0x79; HIGH = 0xFF
